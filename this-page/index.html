$this->page

I have decided to put together a "$this->page", which details all the tech this website uses to run and keep it running. You can find the source code for it here or share your own "$this->page" by sending a pull request to the README here.

Stack:

  • Laravel - I have first used Laravel in development in 2016 and have been in love with it ever since. It is my go-to when developing something for myself.
  • Tailwind CSS - I am not very good at writing front-end code or have found this exercise enjoyable in the past. Tailwind makes CSS a pleasure to write and I find myself learning a lot about CSS in the process.
  • Alpine.js - if CSS isn't my friend, JavaScript has the ability to wrap my brain and squeeze it, until only lemon juice comes out. Something about JS does not work for me, but Alpine.js makes tiny interactions simple again.
  • Sushi - from the creator of Alpine.js, comes another great little package. Sushi is an array-based Eloquent driver and I use it a lot when prototyping the minor features of the page and do not necessarily want to write all the boilerplate yet.
  • laravel-export - This package from Spatie page exports this website as a static HTML, so that it loads fast for you, the reader, and keeps the complexity low for me, the lazy has-been developer.
  • Filament - Filament calls itself "The elegant TALL stack admin for Laravel artisans". It's not fully necessary for this page to run - see reason above - but somehow, it's still much more enjoyable to write blog posts in UI than IDE.
  • Orbit - Why bother with a database, when your plan is to have the code and all its data public? Orbit allows me to still use Eloquent, but having the data persisted in static files.
  • LineAwesome - Font Awesome is, well, awesome, but I've seen it in way too many projects. Line Awesome keeps the same classes as its predecessor but gives your website a fresh look.
  • Unsplash - trust me - I used to take photos for a long time and great photos are hard to take. Try writing blogs and taking pictures. Unsplash makes this process easy for me and we do our best to also give credit to the authors of those shots and hopefully they get rewarded for their work.

Inspiration:

  • Aspire - I have been looking for a beautiful, yet informal and playful page template for a while. Although I changed a lot of parts, I have been heavily inspired by this template. If you're in the market for some web theme, give the creators some love.

Hosting:

  • Github Pages - All my code lives on Github. My personal projects, my open source work and some of my client work. I've used Netlify and Vercel in the past, but simple is better than complex. If it works, don't fix it.
